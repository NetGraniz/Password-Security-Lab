# Password Security Lab

Password Security Lab — веб-приложение для анализа стойкости паролей, расчёта энтропии, оценки времени взлома и генерации безопасных паролей, ключей и лицензий.

🌐 Демо: https://netgraniz.github.io/Password-Security-Lab/

## Возможности

### Анализатор паролей

Проверка пароля в реальном времени:

* Расчёт энтропии в битах
* Определение уровня сложности
* Анализ используемых наборов символов
* Проверка на популярные шаблоны
* Поиск повторяющихся последовательностей
* Рекомендации по усилению безопасности
* Цветовой индикатор стойкости

### Оценка времени взлома

Расчёт приблизительного времени полного перебора для различных сценариев:

* Домашний компьютер
* Игровой ПК
* Современная видеокарта
* Несколько GPU
* Корпоративный вычислительный кластер

Результаты отображаются в удобном формате:

* секунды;
* минуты;
* часы;
* дни;
* годы;
* тысячи лет;
* миллионы лет;
* миллиарды лет.

### Интерактивный график

Визуальное сравнение стойкости пароля относительно:

* рекомендуемого уровня;
* корпоративного уровня;
* высокого уровня безопасности.

### Генератор случайных паролей

Гибкие настройки генерации:

* длина от 4 до 128 символов;
* заглавные буквы;
* строчные буквы;
* цифры;
* специальные символы;
* исключение похожих символов;
* исключение неоднозначных символов.

Готовые пресеты:

* Офисный
* Корпоративный
* Администратор
* Wi-Fi
* Банковский

### Генератор словесных паролей

Создание легко запоминаемых паролей на основе словаря.

Примеры:

```text
Forest-Tiger-Rocket-82
BlueHorse9-RiverMoon
Cloud-Fox-71-Rocket
```

Поддерживаются:

* настройка количества слов;
* выбор разделителя;
* добавление цифр;
* добавление специальных символов;
* различные варианты регистра.

### Генератор ключей и лицензий

Поддержка шаблонов:

```text
AAAAA-AAAAAA-AAAAA
```

Где:

| Символ | Описание                        |
| ------ | ------------------------------- |
| A      | Буква любого регистра или цифра |
| L      | Любая буква                     |
| U      | Заглавная буква                 |
| D      | Цифра                           |
| S      | Специальный символ              |

Примеры:

```text
AAAAA-AAAAAA-AAAAA
LLLLL-DDDDD
UUUU-DDDD-UUUU
```

### Генератор читаемых ключей

Создание удобных для восприятия ключей:

```text
Tiger7-Cloud9-River
BlueFox-82-MoonSky
River-Delta-77-Forest
```

### История и избранное

Сохранение через LocalStorage:

* последних проверок;
* последних генераций;
* избранных паролей;
* пользовательских шаблонов.

Поддерживается экспорт:

* TXT
* CSV

## Запуск локально

Клонируйте репозиторий:

```bash
git clone https://github.com/NetGraniz/Password-Security-Lab.git
```

Перейдите в каталог проекта:

```bash
cd Password-Security-Lab
```

Откройте файл:

```text
index.html
```

или запустите локальный сервер:

```bash
python -m http.server 8000
```

## Для чего создан проект

Password Security Lab предназначен для:

* повышения осведомлённости о безопасности паролей;
* оценки устойчивости к brute-force атакам;
* генерации надёжных паролей;
* генерации лицензионных ключей и шаблонных идентификаторов;
* демонстрации современных возможностей JavaScript без использования серверной части.
